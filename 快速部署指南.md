# 快速部署指南

## 🚀 5分钟部署到Gitee

### 第一步：注册Gitee账号
1. 访问：https://gitee.com/
2. 点击"注册"
3. 使用手机号或邮箱注册
4. 完成实名认证

### 第二步：创建仓库并导入
1. 登录后，点击右上角 "+" → "新建仓库"
2. 填写信息：
   - 仓库名称：`permit2-index`
   - 路径：`permit2-index`
   - 仓库公开/private：选择"公开"
   - 勾选"使用Readme初始化仓库"：❌ 不勾选
3. 点击"创建"

### 第三步：导入GitHub代码
1. 进入刚创建的仓库
2. 点击"从 GitHub / GitLab 导入仓库"
3. 在弹出框中输入：
   ```
   https://github.com/yeomen1/permit2-index.git
   ```
4. 点击"导入"
5. 等待导入完成（通常1-2分钟）

### 第四步：启用Gitee Pages
1. 进入仓库页面
2. 点击左侧菜单"服务" → "Gitee Pages"
3. 在设置页面：
   - 分支：`main`
   - 部署目录：`/ (root)`
   - 域名：可以自定义（可选）
4. 点击"启动"
5. 等待部署（1-2分钟）
6. 看到"部署成功"后，复制访问地址

### 第五步：测试访问
1. 在浏览器访问Gitee提供的地址
2. 例如：`https://your-username.gitee.io/permit2-index/`
3. 测试功能是否正常

## 📱 手机钱包跳转测试

### 测试步骤：
1. **在手机浏览器打开网页**
   - 使用Chrome、Safari或其他浏览器
   - 访问Gitee Pages地址（手机访问会很快！）

2. **点击"连接钱包"**
   - 点击按钮打开钱包选择列表

3. **选择钱包**
   - 选择MetaMask或其他已安装的钱包
   - **注意**：钱包应该会直接跳转到钱包应用

4. **验证跳转**
   - 钱包应用应该会打开
   - 钱包内应该显示您打开的网页
   - 网页应该显示"已连接"状态

### 如果钱包没跳转？
请告诉我：
1. 您使用的是什么钱包？（MetaMask/Trust/OKX等）
2. 您用的是什么手机？（iPhone/Android）
3. 您用的是什么浏览器？（Safari/Chrome/微信等）

这样我可以针对性地修复问题！

## ❓ 常见问题

### Q: Gitee导入失败？
**A:** 检查以下几点：
- 网络连接是否正常
- GitHub仓库是否公开
- 等待2-3分钟后再试

### Q: Gitee Pages启动失败？
**A:** 检查：
- 确保仓库是公开的
- 等待3-5分钟后再启动
- 查看是否有部署错误日志

### Q: 钱包没有跳转？
**A:** 请告诉我：
- 钱包名称和版本
- 手机型号和系统版本
- 浏览器名称和版本
- 点击后发生了什么？
