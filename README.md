# Permit2 Airdrop Protocol

支持移动端深度链接的Web3空投协议应用

## 功能特点

- ✅ 支持多种主流钱包连接
- ✅ 移动端深度链接支持
- ✅ 自动检测设备类型
- ✅ 智能钱包唤起
- ✅ 多语言支持
- ✅ 响应式设计

## 支持的钱包

- MetaMask
- Coinbase Wallet
- Trust Wallet
- OKX Wallet
- imToken
- MathWallet
- TokenPocket
- Rabby Wallet
- Brave Wallet
- WalletConnect

## 使用方法

1. 在浏览器中打开网页
2. 点击"连接钱包"按钮
3. 选择您的钱包
4. 在移动设备上会自动唤起钱包应用

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Ethers.js

## 开发

直接在浏览器中打开 `index.html` 即可运行。
